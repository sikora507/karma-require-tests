# karma-require-tests
## Using Mocha and Chai to test code with RequireJS calls

Install karma globally
```
npm install -g karma-cli
```
Clone the repository
```
cd .\tests
npm install
karma start
```
Output sould be:
```
Chrome 74.0.3729 (Windows 10.0.0): Executed 1 of 1 SUCCESS (0.006 secs / 0 secs)
TOTAL: 1 SUCCESS
```
